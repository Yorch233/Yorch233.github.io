# Yorch233.github.io

Personal site and paper demo pages built with a modern front-end toolchain.

## Tech Stack
- Vite (dev server & build)
- Vue 3 (Composition API)
- JavaScript (ESM)

## Project Structure
- src/views: pages and view-level content
- src/components: shared components (audio compare, message box, etc.)
- src/assets: static assets and paper audio samples
- src/modules: router and state management

## Requirements
- Node.js >= 18
- npm >= 9

## Development
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build for production: `npm run build`

## Deployment
This site can be deployed to GitHub Pages. The build output is generated by Vite and can be published as static files.
